Chapter summary

Presentation: 23FX

  • A vector has a magnitude and direction.

  • Vectors can be used to represent many physical quantities that have a magnitude and direction, like forces.

  • Vectors may be represented as arrows where the length of the arrow indicates the magnitude and the arrowhead indicates the direction of the vector.

  • Vectors in two dimensions can be drawn on the Cartesian plane.

  • Vectors can be added graphically using the head-to-tail method or the tail-to-tail method.

  • A closed vector diagram is a set of vectors drawn on the Cartesian using the tail-to-head method and that has a resultant with a magnitude of zero.

  • Vectors can be added algebraically using Pythagoras' theorem or using components.

  • The direction of a vector can be found using simple trigonometric calculations.

  • The components of a vector are a series of vectors that, when combined, give the original vector as their resultant.

  • Components are usually created that align with the Cartesian coordinate axes. For a vector F ⃗ \vec{F} that makes an angle of θ \theta with the positive x x-axis the x x-component is R ⃗ x = R cos ⁡ ( θ ) \vec{R}_x=R\cos(\theta) and the y y-component is R ⃗ y = R sin ⁡ ( θ ) \vec{R}_y=R\sin(\theta).

Four forces act simultaneously at a point, find the resultant if the forces are:

  • F ⃗ 1 \vec{F}_{1} = 2,3 \text{2,3}~ N \text{N} in the positive x x-direction

  • F ⃗ 2 \vec{F}_{2} = 4,9 \text{4,9}~ N \text{N} in the positive y y-direction

  • F ⃗ 3 \vec{F}_{3} = 4,3 \text{4,3}~ N \text{N} in the negative y y-direction

  • F ⃗ 4 \vec{F}_{4} = 3,1 \text{3,1}~ N \text{N} in the negative y y-direction

We note that we have more than two vectors so we must first find the resultant in the x x-direction and the resultant in the y y-direction.

In the x x-direction we only have one vector and so this is the resultant.

In the y y-direction we have three vectors. We can add these algebraically to find R ⃗ y \vec{R}_{y}:

F ⃗ 2 = +4,9  N F ⃗ 3 = − 4,3  N F ⃗ 4 = − 3,1  N \begin{align*} \vec{F}_{2} & = \text{+4,9}\text{ N} \\ \vec{F}_{3} & = -\text{4,3}\text{ N} \\ \vec{F}_{4} & = -\text{3,1}\text{ N} \end{align*}

Thus, the resultant force is:

F ⃗ 2 + F ⃗ 3 + F ⃗ 4 = ( 4,9 ) + ( − 4,3 ) + ( − 3,1 ) = − 2,3 \begin{align*} \vec{F}_{2} + \vec{F}_{3} + \vec{F}_{4} & = (\text{4,9}) + (-\text{4,3}) + (-\text{3,1}) \\ & = -\text{2,3} \end{align*}

Now we use R ⃗ x \vec{R}_{x} and R ⃗ y \vec{R}_{y} to find the resultant.

We note that the triangle formed by R ⃗ x \vec{R}_{x}, R ⃗ y \vec{R}_{y} and the resultant vector is a right-angle triangle. We can thus use the Theorem of Pythagoras to determine the length of the resultant. Let R R represent the length of the resultant vector. Then: F x 2 + F y 2 = R 2   Pythagoras' theorem ( 2,3 ) 2 + ( − 2,3 ) 2 = R 2 R = 3,25  N \begin{align*} F_x^{2} + F_y^{2} &= R^{2}\ \textrm{Pythagoras' theorem}\\ (\text{2,3})^{2} + (-\text{2,3})^{2} &= R^{2}\\ R &= \text{3,25}\text{ N} \end{align*}

To determine the direction of the resultant force, we calculate the angle α between the resultant force vector and the positive x x-axis, by using simple trigonometry: tan ⁡ α = opposite side adjacent side tan ⁡ α = 2,3 2,3 α = tan − 1 ⁡ ( 1 ) α = 45 ° \begin{align*} \tan\alpha &= \frac{\textrm{opposite side}}{\textrm{adjacent side}} \\ \tan\alpha &= \frac{\text{2,3}}{\text{2,3}} \\ \alpha &= \tan^{-1}(\text{1}) \\ \alpha &= \text{45}\text{°} \end{align*}

The resultant force is then 3,25 \text{3,25}~ N \text{N} at 45 \text{45} ° \text{°} to the x x-axis.
